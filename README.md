Automação Mobile - Calculadora Android

## Sobre o projeto

Este projeto foi desenvolvido como parte da **Lista de Exercícios 11** do curso de Automação Mobile da Iterasys.

O objetivo é automatizar operações básicas utilizando o aplicativo nativo **Calculadora** de um dispositivo Android, empregando o **Appium** juntamente com o **WebdriverIO**.

---

## Objetivo

Automatizar a execução e validação das seguintes operações matemáticas:

- ✅ Soma
- ✅ Subtração
- ✅ Multiplicação
- ✅ Divisão

Cada operação é realizada utilizando dois números previamente definidos e o resultado é validado automaticamente.
